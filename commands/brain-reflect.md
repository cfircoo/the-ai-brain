---
name: brain-reflect
description: AI self-review and pattern analysis
---

Run the `reflect` skill to analyze recent sessions and propose improvements.

Optionally accepts a time range argument (e.g., "7 days", "30 days"). Defaults to the last 7 days. Reads session logs, corrections, decisions, and daily notes within the range. Identifies recurring themes, repeated corrections, user working patterns, and chronically deferred tasks. Proposes rule improvements and process changes. Saves a reflection report to Machine/.
