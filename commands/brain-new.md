Run the `new` skill to triage raw input into the vault.

Accept the user's unstructured input (tasks, notes, ideas, logs -- all mixed together), parse it into distinct items, classify each by type, and route to the correct vault location with proper frontmatter, tags, and wikilinks. Log what was processed in today's daily note.

If no input is provided alongside this command, ask the user what they want to capture.
