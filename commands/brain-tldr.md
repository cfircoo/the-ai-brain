Run the `tldr` skill to generate a concise summary.

If no argument is given, summarize the current conversation session. If a file path is provided, summarize that file. If a topic is provided, search the vault for relevant notes and synthesize a summary across sources. Always highlight key decisions and next steps. Offer to save session summaries to Machine/Session-Logs/.
