Run the `debrief` skill to perform the end-of-session ritual.

Summarize what was accomplished in this session, write a structured session log to Machine/Session-Logs/, update all memory files (context-cache, entities, decisions, corrections as needed), identify incomplete tasks for carryover to tomorrow, update .brain/state.json, and append a session summary to today's daily note.
