description: Vault health check and consistency report
---

Run the `vault-audit` skill to perform a comprehensive vault health check.

Count files by directory, find orphaned notes with no incoming wikilinks, check frontmatter consistency across all files, identify stale files not modified in 90+ days, analyze tag usage patterns, and verify the vault structure matches brain.yaml. Present a formatted health report and offer actionable fixes.
